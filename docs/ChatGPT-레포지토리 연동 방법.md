### 불가능한 것

반대로 이런 방식은 어렵습니다.

- 제가 레포지토리를 **항상 백그라운드에서 실시간 감시**

즉, **메모리는 프로젝트 방향성, 선호 구조, 이전 결정사항을 기억하는 용도**에 가깝고, **실제 최신 코드는 매번 GitHub에서 다시 읽어 확인하는 방식**이 안전합니다.
