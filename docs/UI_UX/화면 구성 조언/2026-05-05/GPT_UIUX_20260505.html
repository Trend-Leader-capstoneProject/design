업로드해주신 기획 수정안을 보면 핵심 방향이 **“최신 트렌드 전체 제공” → “내 관심사 기준으로 최신 흐름을 골라보는 개인화 트렌드 앱”**으로 정리되었습니다. 특히 기획 수정안에서는 대상 타깃을 “패션, 뷰티, 음식, 게임 등 자신의 관심 분야 최신 트렌드만 골라 보고 싶은 10~20대 사용자”로 좁히고, 핵심 기능을 관심사 기반 트렌드 탭, 보조 기능을 최신 이슈 탭으로 나눈 점이 중요합니다.

이 기준이면 화면은 단순히 “랭킹 리스트”만 늘리는 것보다, 아래처럼 관심사 선택 → 개인화 홈 → 트렌드 상세 분석 → 최신 이슈 탐색 → 저장/마이페이지 흐름으로 확장하는 것이 좋습니다.

1. 전체 화면 구조 제안

현재 기획을 기준으로 하면 앱의 주요 화면은 이렇게 나누는 것이 자연스럽습니다.

1. 스플래시 / 시작 화면
2. 로그인 / 회원가입 화면
3. 관심 카테고리 선택 화면
4. 메인 홈 화면
5. 관심사 기반 트렌드 탭
6. 트렌드 상세 화면
7. 최신 이슈 탭
8. 검색 / 필터 화면
9. 저장한 트렌드 화면
10. 마이페이지 / 관심사 재설정 화면

다만 캡스톤 MVP 기준으로는 너무 많은 화면을 한 번에 구현하기보다, 우선순위를 나누는 것이 좋습니다.

MVP 우선 구현 화면

1. 로그인 / 회원가입
2. 관심 카테고리 선택
3. 메인 홈
4. 관심사 기반 트렌드 리스트
5. 트렌드 상세
6. 최신 이슈 탭
7. 마이페이지

후순위 확장 화면

1. 저장한 트렌드
2. 알림 설정
3. 검색 기록
4. 관심 키워드 직접 등록
5. AI 챗봇형 트렌드 질문
6. 커뮤니티 / 의견 공유

2. 추천 앱 구조

기획안의 핵심은 관심사 기반 개인화이므로, 하단 탭은 너무 많지 않게 4개 정도가 적당합니다.

[홈] [최신이슈] [저장] [마이]

또는 조금 더 기획 의도를 강조하면:

[내 트렌드] [전체 이슈] [저장] [마이]

개인적으로는 발표와 기획 설명에서 더 명확한 이름은 아래입니다.

[내 트렌드] [최신 이슈] [저장] [마이]

이렇게 하면 사용자가 앱에 들어오자마자 “이 앱은 전체 뉴스 앱이 아니라 내 관심사 중심 앱이구나”라고 이해하기 쉽습니다.

색상 역할

메인 컬러: 연보라 / 블루 계열
강조 컬러: 진한 파랑
배경 컬러: 거의 흰색에 가까운 연회색
카드 컬러: 흰색 또는 아주 연한 보라
위험/상승 표시: 빨강보다는 오렌지나 핑크 정도

너무 많은 색을 쓰면 트렌드 앱이 아니라 쇼핑몰처럼 보일 수 있으니, 카테고리별 색상은 은은하게만 쓰는 것이 좋습니다.

6. 기능상세명세서로 확장하기 좋은 화면

이제 기획서를 기능상세명세서로 확장해야 한다면, 가장 먼저 아래 4개 화면을 상세화하는 것이 좋습니다.

1. 관심 카테고리 선택 화면
2. 관심사 기반 트렌드 메인 화면
3. 트렌드 상세 화면
4. 최신 이슈 탭 화면

예를 들어 “관심 카테고리 선택 화면”은 이렇게 명세할 수 있습니다.

기능명: 관심 카테고리 선택

목적:
사용자가 관심 있는 분야를 선택하여 이후 메인 화면에서 개인화된 트렌드를 제공받도록 한다.

사용자 동작:
- 사용자는 패션, 뷰티, 게임, 음식 등의 카테고리 중 3개 이상 선택한다.
- 선택 완료 버튼을 누른다.

처리 로직:
- 선택한 카테고리 ID 목록을 서버로 전송한다.
- 서버는 사용자 ID와 카테고리 ID를 매핑하여 저장한다.
- 저장 완료 후 메인 화면으로 이동한다.

데이터 저장:
- user_interest 테이블에 user_id, category_id를 저장한다.

예외 처리:
- 3개 미만 선택 시 “관심 분야를 3개 이상 선택해주세요.” 메시지를 표시한다.
- 서버 저장 실패 시 “관심사 저장에 실패했습니다. 다시 시도해주세요.” 메시지를 표시한다.

이런 방식으로 화면별 기능상세명세서를 이어가면 발표 자료와 구현 설계가 자연스럽게 연결됩니다.

7. 최종 추천 화면 구성안

가장 현실적인 MVP 화면 구성은 아래입니다.

[필수]
1. 로그인 / 회원가입
2. 관심 카테고리 선택
3. 내 트렌드 홈
4. 카테고리별 트렌드 리스트
5. 트렌드 상세
6. 최신 이슈
7. 마이페이지

[선택]
8. 저장한 트렌드
9. 알림 설정
10. 검색 화면

현재 기획 수정안의 방향을 살리려면 화면 이름도 아래처럼 잡는 것이 좋습니다.

메인 화면 → 내 트렌드
인기 검색어 → 관심사 트렌드
상세 페이지 → 트렌드 분석
전체 탭 → 최신 이슈

이렇게 바꾸면 서비스의 정체성이 더 선명해집니다.

핵심은 **“요즘 뭐가 유행하지?”**가 아니라, **“내가 관심 있는 분야에서는 지금 뭐가 뜨고 있지?”**를 해결하는 앱으로 화면을 구성하는 것입니다.
