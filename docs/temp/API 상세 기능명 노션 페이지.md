## 8. 예외 응답

| 상태 코드 | 상황 | 메시지 예시 | 프론트 처리 |
| --- | --- | --- | --- |
| 400 | 잘못된 요청 | 입력값이 올바르지 않습니다. | Alert 또는 Toast 표시 |
| 401 | 인증 실패 | 로그인이 필요합니다. | 로그인 화면으로 이동 |
| 403 | 권한 없음 | 접근 권한이 없습니다. | 권한 오류 안내 |
| 404 | 데이터 없음 | 요청한 데이터를 찾을 수 없습니다. | 빈 상태 화면 표시 |
| 500 | 서버 오류 | 서버 오류가 발생했습니다. | 재시도 안내 |
