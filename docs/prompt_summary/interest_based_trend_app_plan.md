# 관심사 기반 트렌드 추천 앱 간단 기획서

## 1. 서비스 개요

본 서비스는 사용자가 자신의 관심 분야에 맞는 최신 트렌드를 빠르게 탐색할 수 있도록 돕는 **관심사 기반 개인화 트렌드 큐레이션 앱**이다.

기존의 트렌드 정보는 SNS, 검색 플랫폼, 커뮤니티 등 여러 채널에 흩어져 있어 사용자가 직접 찾아보기 어렵고, 관심 없는 이슈까지 함께 확인해야 하는 불편함이 있다.  
이에 본 서비스는 사용자가 선택한 관심 카테고리를 중심으로 관련 트렌드를 우선 제공하고, 보조적으로 최신 이슈 탭을 통해 전체적인 유행 흐름도 확인할 수 있도록 한다.

---

## 2. 기획 배경

최근 트렌드는 패션, 뷰티, 게임, 음식, 엔터테인먼트 등 다양한 분야에서 빠르게 변화하고 있다.  
하지만 사용자는 여러 플랫폼을 직접 확인해야 하며, 단순 인기 검색어만으로는 해당 트렌드가 왜 주목받는지 이해하기 어렵다.

따라서 본 프로젝트는 단순히 “최신 트렌드를 보여주는 앱”이 아니라, 사용자의 관심사에 맞춰 필요한 트렌드를 선별하고, AI 분석을 통해 트렌드의 핵심 내용과 관련 키워드를 이해할 수 있도록 돕는 것을 목표로 한다.

---

## 3. 핵심 타깃

### 주 타깃

패션, 뷰티, 게임, 음식 등 자신의 관심 분야에서 어떤 주제가 유행하고 있는지 빠르게 파악하고 싶은 사용자

### 보조 타깃

요즘 전체적으로 유행하는 주제와 사회적 이슈를 빠르게 확인하고 싶은 사용자

### 타깃 사용자 예시

- 여러 플랫폼을 직접 확인하지 않고 관심 분야 트렌드를 한 화면에서 보고 싶은 사용자
- 관심 없는 이슈보다 자신이 선택한 분야의 트렌드를 우선적으로 보고 싶은 사용자
- 유행 키워드뿐만 아니라 관련 키워드와 유행 이유까지 간단히 파악하고 싶은 사용자
- 최신 이슈 탭을 통해 관심사 밖의 전체적인 흐름도 함께 확인하고 싶은 사용자

---

## 4. 서비스 목적

본 서비스의 목적은 사용자가 관심 분야의 최신 트렌드를 빠르게 확인하고, AI 기반 분석을 통해 해당 트렌드의 의미와 흐름을 쉽게 이해할 수 있도록 지원하는 것이다.

이를 통해 사용자는 다음과 같은 가치를 얻을 수 있다.

1. 관심 분야 중심으로 트렌드를 선별해 볼 수 있다.
2. 여러 플랫폼에 흩어진 트렌드 정보를 한곳에서 확인할 수 있다.
3. 단순 키워드 순위가 아니라 유행 이유와 관련 키워드를 함께 파악할 수 있다.
4. 최신 이슈 탭을 통해 개인 관심사 밖의 전체적인 흐름도 확인할 수 있다.

| 관심사 불일치 | 전체 인기 이슈 중 사용자 관심사와 관련 없는 정보가 많음 |

## 7. 주요 기능

### 7.1 관심사 선택 기능

사용자는 회원가입 또는 초기 설정 단계에서 관심 카테고리를 선택한다.

예시 카테고리:

- 패션
- 뷰티
- 게임
- 음식
- 엔터테인먼트
- IT/디지털

선택된 관심사는 사용자별 추천 기준으로 활용된다.

---

### 7.2 관심사 기반 트렌드 조회

사용자가 선택한 관심 카테고리를 기준으로 관련 트렌드 목록을 제공한다.

예시:

- 게임 관심 사용자: 인기 게임, 업데이트 이슈, 게임 밈, e스포츠 이슈
- 음식 관심 사용자: 유행 메뉴, 맛집 키워드, 신제품 음식 트렌드
- 패션 관심 사용자: 유행 스타일, 브랜드 이슈, 계절별 코디 키워드

---

### 7.3 최신 이슈 탭

사용자의 관심사와 무관하게 전체적으로 주목받는 최신 이슈를 확인할 수 있는 보조 기능이다.

최신 이슈 탭은 서비스의 핵심 기능이라기보다, 사용자가 개인 관심사 밖의 사회적 흐름을 빠르게 파악할 수 있도록 돕는 보조 화면으로 설계한다.

---

### 7.4 트렌드 상세 조회

사용자가 특정 트렌드를 선택하면 상세 화면에서 다음 정보를 확인할 수 있다.

- 트렌드 키워드
- 간단 설명
- 유행 이유
- 관련 키워드
- 관련 플랫폼 또는 출처
- 저장 또는 북마크 기능

---

### 7.5 AI 기반 트렌드 분석

AI는 단순히 “분석한다”는 표현이 아니라, 다음과 같은 구체적인 역할을 수행한다.

- 트렌드 요약
- 유행 배경 설명
- 관련 키워드 추출
- 사용자 관심사와의 연관성 판단
- 유사 트렌드 추천

초기 버전에서는 AI 기능을 트렌드 요약과 관련 키워드 제공 중심으로 제한하고, 이후 사용자 활동 데이터를 기반으로 추천 정확도를 개선하는 방향으로 확장할 수 있다.

## 10. 데이터 처리 방향

### Frontend

- React Native 기반 모바일 화면 구성
- 사용자의 관심사 선택, 트렌드 목록 조회, 상세 조회 요청 처리
- API 응답 데이터를 카드형 리스트, 탭, 상세 화면 형태로 표시

### Backend

- Python FastAPI 기반 API 서버 구성
- 사용자 정보, 관심사, 트렌드 데이터, 저장 목록 처리
- AI 분석 요청 및 결과 저장/조회 처리

### Database

MySQL 또는 MariaDB 사용을 가정한다.

예시 테이블:

| 테이블 예시 | 주요 필드 예시 | 설명 |
|---|---|---|
| users | user_id, email, password_hash, nickname | 사용자 기본 정보 |
| interest_categories | category_id, category_name | 관심 카테고리 정보 |
| user_interests | user_id, category_id | 사용자별 관심사 매핑 |
| trends | trend_id, keyword, category_id, source, rank, collected_at | 수집된 트렌드 정보 |
| trend_ai_analysis | analysis_id, trend_id, summary, reason, related_keywords | AI 분석 결과 |
| user_bookmarks | bookmark_id, user_id, trend_id, created_at | 사용자 저장 트렌드 |

※ 실제 테이블명과 필드명은 구현 과정에서 변경될 수 있다.

## 13. 발표용 핵심 문장

본 서비스는 단순히 최신 트렌드를 보여주는 앱이 아니라, 사용자의 관심 분야에 맞는 최신 트렌드를 선별해 제공하는 개인화 트렌드 큐레이션 앱이다.

핵심 타깃은 패션, 뷰티, 게임, 음식 등 자신의 관심 분야에서 유행하는 주제를 빠르게 확인하고 싶은 사용자이며, 최신 이슈 탭은 개인 관심사 밖의 전체적인 흐름을 보조적으로 확인하기 위한 기능이다.

AI는 트렌드 요약, 유행 이유 설명, 관련 키워드 추출을 통해 사용자가 단순 키워드 순위를 넘어 트렌드의 맥락을 이해할 수 있도록 돕는다.

## 14. 정리

본 프로젝트의 핵심 방향은 다음과 같다.

1. 서비스 정체성: 관심사 기반 개인화 트렌드 큐레이션 앱
2. 주 타깃: 자신의 관심 분야 최신 트렌드를 빠르게 확인하고 싶은 사용자
3. 보조 기능: 전체 최신 이슈 확인 탭
4. 핵심 기능: 관심사 선택, 관심사 기반 추천, 트렌드 상세 조회, AI 요약
5. 확장 방향: 사용자 활동 기반 추천 고도화, 알림, 커뮤니티, 트렌드 분석 그래프
